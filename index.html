Problem

왜 지금, 이 서비스가 필요한가

셀프 인테리어 카페에서 반복되는 질문의 80%는 "이거 순서 맞나요?", "이 견적 적정한가요?", "이 단계에서 뭘 확인해야 하나요?" 세 가지로 수렴한다.

Screen Architecture

핵심 화면 구성

공정 중심으로 계획+기록을 통합, 각 공정에서 전문가 매칭까지 원스톱

[홈]
  ├─ 내 프로젝트 (진행중인 인테리어)
  │   ├─ 공정 타임라인 (현재 단계 하이라이트)
  │   ├─ 오늘의 체크리스트
  │   └─ 예산 현황 요약

[공정] ← 기존 '계획'+'기록' 통합
  ├─ 공정 리스트 (추가/삭제/순서변경)
  │   └─ 각 공정 상세
  │       ├─ 체크리스트 & To-Do
  │       ├─ 기록 (사진/영상/메모)
  │       ├─ 전후 비교
  │       ├─ 전문가 찾기 (견적/매칭/예약)
  │       │    ├─ 전문가 프로필 & 포트폴리오
  │       │    ├─ 견적 비교
  │       │    └─ 시공 리뷰 & 평점
  │       ├─ 필요 자재 목록
  │       └─ 비용 (예상 vs 실제)
  ├─ 비용 계산기 (전체 요약)
  ├─ 캘린더
  ├─ 도면
  └─ 블로그 글 생성 (기록 기반)

[쇼핑] ← 자재/커머스 특화
  ├─ 자재 카테고리 브라우징
  ├─ 인기 자재 & 추천
  ├─ 자재 가격 비교
  ├─ 공정별 필요 자재 추천
  ├─ 도면 연동 수량 자동 계산
  ├─ 장바구니 & 구매
  └─ 주문/배송 관리

[AI 도우미]
