Problem

왜 지금, 이 서비스가 필요한가

인테리어를 직접 해보려는 사람은 폭발적으로 늘고 있지만, 공정별 전문가 연결을 도와주는 전문 도구는 없다. 정보는 파편화되어 있고, 비용은 불투명하며, 공정 관리는 여전히 수기에 의존한다.

CORE INSIGHT

셀프 인테리어 카페에서 반복되는 질문의 80%는 "이거 순서 맞나요?", "이 견적 적정한가요?", "이 단계에서 뭘 확인해야 하나요?" 세 가지로 수렴한다. 이 세 가지를 즉시 해결하는 도구가 곧 시장 진입의 핵심이다.

Go-to-Market

시장 진입 전략

커뮤니티 침투 → 콘텐츠 바이럴 루프 → 양면 네트워크 효과

Phase 1: 0~6개월

커뮤니티 침투

네이버 셀프인테리어 카페에 비용 계산기/체크리스트를 무료 도구로 바이럴. 블로그 SEO로 "셀프 인테리어 비용", "인테리어 순서" 키워드 점령. 초기 유저 1만명 확보.

Phase 2: 6~12개월

콘텐츠 바이럴 루프

유저가 만든 블로그 글 → 자연 검색 유입 → 앱 다운로드. 완료된 프로젝트 공유 → 새 유저가 레퍼런스로 활용. 자기 증식하는 콘텐츠 엔진.

Phase 3: 12개월~

양면 네트워크 효과

전문가가 많을수록 유저 유입 ↑ → 유저가 많을수록 전문가 유입 ↑. 양면 네트워크 효과로 경쟁자 진입 장벽 구축.

Growth Plan

채널별 유저 확보 전략

서비스만으로는 유저가 오지 않는다. 인테리어를 준비하는 사람이 이미 모여 있는 곳에서 시작한다.

ORGANIC

검색 SEO + 블로그 콘텐츠

인테리어 준비자는 반드시 검색한다. "셀프 인테리어 비용", "인테리어 공정 순서", "타일 시공 단가" 등 롱테일 키워드를 선점한다.

  • 공정별 비용 가이드 시리즈 (30편+) → 네이버/구글 상위 노출
  • 비용 계산기를 웹 버전으로 공개 → 결과 페이지에서 앱 전환 유도
  • 유저가 생성한 블로그 글 → 자연 검색 유입의 선순환
  • 월 목표: 검색 유입 5,000+ → 앱 전환 15%
COMMUNITY

인테리어 커뮤니티 침투

네이버 카페 "셀프 인테리어"(40만+), "인테리어 정보"(25만+) 등 이미 타겟 유저가 모여 있는 커뮤니티에서 실질적 가치를 제공한다.

  • 비용 계산 결과 공유 → "이 앱 뭐예요?" 자연 바이럴
  • 공정 체크리스트 PDF 무료 배포 → 앱 다운로드 CTA
  • 인테리어 후기 작성 이벤트 (앱 내 블로그 생성 기능 체험)
  • 월 목표: 카페/커뮤니티 유입 3,000+
PAID

퍼포먼스 마케팅 (Phase 2~)

PMF 확인 후 유닛 이코노믹스가 맞는 채널에만 집행. CAC < LTV의 3분의 1 기준.

  • 네이버 검색 광고: "인테리어 견적", "셀프 인테리어" 키워드
  • 인스타그램/유튜브: 전후 비교 사진/영상 소재 (CTR 높음)
  • 리타겟팅: 웹 비용 계산기 사용 후 이탈한 유저
  • CAC 목표: 5,000원 이하 (구독 전환 기준 15,000원 이하)
PARTNERSHIP

전략적 제휴

인테리어를 시작하는 순간에 자연스럽게 노출되는 접점을 확보한다.

  • 부동산 앱 제휴: 입주 예정자에게 인테리어 계획 도구 추천
  • 자재 브랜드 협업: LX하우시스, 한샘 등 자재 카탈로그 연동
  • 인테리어 업체 B2B: 고객 관리 도구로 제공 → 역방향 유저 유입
  • 아파트 입주 커뮤니티 제휴 (입주 단톡방 등)
